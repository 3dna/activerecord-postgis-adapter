activerecord-postgis-adapter

ActiveRecord connection adapter for PostGIS, based on postgresql and rgeo

Download this project as a .zip file Download this project as a tar.gz file

The PostGIS ActiveRecord Adapter extends the standard PostgreSQL adapter to provide support for the spatial extensions provided by PostGIS, using the RGeo library to represent spatial data in Ruby. It provides:

  • Extensions to the migration syntax for creating spatial columns and indexes
  • Attributes using RGeo spatial data types
  • Spatial queries
  • PostGIS versions of some of the ActiveRecord rake tasks

See the README for important information on installation and usage.
